**Summary**
* **Years:** 2018-2019
* **Languages:** Ruby on Rails, PHP, MySQL
* **Frameworks:** CollectiveAccess, Apache Solr
* **Description:** Primary frontend developer for University Art Collection website at Cal Poly SLO.

I built the frontend for the [University Art Collection](https://artcollection.calpoly.edu) website in 2018, which hosts a database of the art owned by Cal Poly SLO. The website uses [CollectiveAccess](https://www.collectiveaccess.org)&mdash;open-source software for cataloguing and publishing museum and archival collections, running on PHP/MySQL. I wrote the frontend using Ruby on Rails.

Pieces in the art collection are stored in a MySQL database and indexed into Apache Solr. The public search interface uses Blacklight, a Rails engine commonly used by libraries and museums for digital collections.
