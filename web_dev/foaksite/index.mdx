**Summary**
* **Years:** 2026-present
* **Languages:** React, Tailwind
* **Frameworks:** Astro, Sanity
* **Description:** I maintain the [website](https://fiveofakindbrass.com) for my brass quintet, Five of a Kind. It is a React (Astro) frontend and uses Sanity for the events database.

I play trumpet in a brass quintet, [Five of a Kind](https://fiveofakindbrass.com). To expand our presence and land more gigs, we wanted a website. So, I put together a simple React site using Astro. Since I wanted the rest of the quintet to be able to edit the "upcoming events" portion of the site without programming knowledge, I implemented a Sanity backend.
